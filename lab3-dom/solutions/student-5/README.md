# Расширение для браузера Chrome
Добавляет кнопку для изменения стиля сайта kai.ru

# KAI Sepia Style - Chrome Extension

Плагин для браузера, который меняет стиль сайта kai.ru на Sepia / Old Paper: Коричневый фон, темно-коричневый текст

#Функциональность

- Добавляет кнопку переключения стиля на главную страницу
- Изменяет внешний вид сайта на стиль "старой бумаги"
- Сохраняет состояние стиля между перезагрузками страницы
- Работает на главной странице и страницах навигации

# Установка

# Для Chrome

1. Скачайте все файлы расширения в одну папку
2. Откройте браузер и перейдите в "Настройки", "Расширения"
3. Включите "Режим разработчика" 
4. Нажмите "Загрузить распакованное расширение"
5. Выберите папку с файлами плагина

# Использование

1. Перейдите на сайт [kai.ru](https://kai.ru)
3. Нажмите на кнопку для включения/выключения сепиа стиля
4. Статус кнопки показывает текущее состояние стиля

# Изменяемые стили


Фон страницы - цвет старой бумаги (#f8f4e8)
Цвет текста - коричневый как чернила (#5a4a42)
Шрифты - Times New Roman/Georgia для старославянского вида
Фон блоков - светлый пергамент (#f9f5eb)
Границы - стилизованные рамки (#d4c4a8)
Заголовки - выделенные цветом и шрифтом
Навигация - стилизованные ссылки
Кнопки - соответствующие общей цветовой схеме
